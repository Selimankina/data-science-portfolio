# Определение возраста по фотографии

## Технологии

Python • Pandas • NumPy • Matplotlib • Seaborn • TensorFlow / Keras • ResNet50

## Описание проекта

Разработана модель компьютерного зрения для определения приблизительного возраста человека по изображению лица.

Проект демонстрирует применение глубокого обучения для анализа фотографий покупателей в ритейле.

## Цель

Построить модель, предсказывающую возраст по фото с точностью 8 лет.

**Метрика качества:** MAE ≤ 8 лет

## Данные

* 7 591 изображение лиц
* Возраст: от 1 до 100 лет
* `labels.csv` содержит имя файла и реальный возраст
* Пропуски и дубликаты отсутствуют

## Подход к решению

Использован transfer learning на основе предобученной сети ResNet50.

Обучение проходило в два этапа:

1. Обучение верхних слоёв при замороженной сверточной базе
2. Fine-tuning всей сети

## Результат

**MAE = 7.60 лет** — модель достигла требуемой точности.

## Ограничения

* Дисбаланс возрастных групп
* Влияние освещения и ракурсов
