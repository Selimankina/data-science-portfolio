# Прогнозирование оттока клиентов телеком-компании

## Используемые технологии

Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, CatBoost, SHAP

------------------------------------------------------------------------

## Обзор проекта

В данном проекте решается задача прогнозирования оттока клиентов
телеком-компании с использованием методов машинного обучения.

Цель проекта — выявить клиентов с высокой вероятностью расторжения
контракта и предоставить бизнесу инструменты для удержания клиентов.

Проект включает полный цикл анализа данных:

• предобработку данных\
• исследовательский анализ (EDA)\
• визуализацию распределений и зависимостей\
• обучение моделей машинного обучения\
• подбор гиперпараметров\
• сравнение моделей\
• оценку качества на тестовой выборке\
• интерпретацию результатов\
• анализ важности признаков и SHAP-анализ

------------------------------------------------------------------------

## Бизнес-задача

Отток клиентов приводит к финансовым потерям и росту затрат на
маркетинг.

Задачи проекта:

• предсказать вероятность оттока клиентов\
• определить ключевые факторы оттока\
• дать рекомендации по удержанию клиентов

Целевая метрика качества: **ROC-AUC ≥ 0.85**

------------------------------------------------------------------------

## Описание данных

Датасет содержит информацию о клиентах:

• параметры контрактов\

## Использованные модели

### Логистическая регрессия

Базовая интерпретируемая линейная модель.

### Случайный лес

Ансамблевая модель для выявления нелинейных зависимостей.

### CatBoost (лучшая модель)

Градиентный бустинг с поддержкой категориальных признаков.

Преимущества:

• минимальная предобработка\
• высокая точность на табличных данных\
• встроенные инструменты интерпретации

## Интерпретация модели

### Важность признаков

Наибольшее влияние на прогноз оказывают:

1.  Длительность контракта\
2.  Ежемесячные платежи\
3.  Тип контракта\
4.  Способ оплаты\
5.  Количество подключенных услуг

### SHAP-анализ

SHAP позволяет:

• объяснять индивидуальные прогнозы\
• определять вклад каждого признака\
• повышать прозрачность модели

------------------------------------------------------------------------

## Бизнес-выводы

Модель может применяться для:

• выявления клиентов с высоким риском ухода\
• приоритизации программ удержания\
• оптимизации маркетинговых затрат

Ключевые факторы оттока:

• контракты средней длительности\
• высокая стоимость услуг\
• гибкие тарифные планы
